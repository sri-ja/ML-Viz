I have a secret! A computer does not see an image exactly like we do. What a computer sees is actually a box of pixels, or small boxes, each of which has a specific color. This color can be represented in different systems like the RGB system.

Those smaller boxes come together to make up a bigger picture or image that we see. You can use the 'See pixels' button to find out what color each box is.

color

Now, for the computer to be able to recognize it, it uses different kinds of algorithms and performs different kinds of actions on the image. One of them is known as the edge detection algorithm which uses the Sobel Operator. I know those words sound big and scary but I promise they are not scary.

What it basically does is it analyzes and works with each of the pixel of the image in order to help detect the presence of edges (or borders) in the image. That way we go from that pretty pink flower to this.

I know that image doesn't look as pretty as the original, but our computer friend finds it really helpful. So what it does now, is that it takes parts of the image and starts analyze them with already existing images in a data set. Now what is a data set you ask?

To put it simply, it is a collection of data (images in this scenario) which we can feed to the amazing machine learning model in order to teach it about the different objects in the world, help it recognize it.
